# IHS Anonymized Encounter Dashboard

## Overview
This Project is meant to show in an anonymized way all the work done on IHS ( Integrated Health Services Dashboard).

## Data and Anonymization
- Data Anonymization was perfomred with Presidio and Python.
- Tables used include: CDV, patient, demographic, user_info.

## Power BI Dashboard
- The Encounters Dashboard allowed to get rid of the need to quarterly extract manually all the encounters realted to Doctors and Nurses, and instead it uses Fiscal Date Slicers to handle the encounter numbers by any fiscal period, Yearly, quarterly or montly.
